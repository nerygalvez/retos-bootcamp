# Roman to Integer

## Complejidad:
Fácil

## Descripción
Los números romanos están representados por 7 diferentes simbolos: I, V, X, L, C, D y M.

Valores:
```
Símbolo       Valor
I             1
V             5
X             10
L             50
C             100
D             500
M             1000
```
## Ejemplos:

**Ejemplo 1:**
```
Input: s = "III"
Output: 3
Explanation: III = 3.
```

**Ejemplo 2:**
```
Input: s = "LVIII"
Output: 58
Explanation: L = 50, V= 5, III = 3.
```

**Ejemplo 3:**
```
Input: s = "MCMXCIV"
Output: 1994
Explanation: M = 1000, CM = 900, XC = 90 and IV = 4.
```
